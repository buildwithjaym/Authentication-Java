# Authentication System

This project is a simple authentication system developed using Java in NetBeans IDE 12.2. The system uses MySQL as the database to manage user credentials.

## Technologies Used

- **Java** (developed in NetBeans IDE 12.2)
- **MySQL** for database management

## Features

- User registration
- User login authentication

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the project in NetBeans IDE 12.2.
3. Set up the MySQL database:
   - Import the provided database schema.
   - Update database connection settings in the project if necessary.
4. Run the project and test the authentication features.
